# TeamUp Russian Product Glossary

## Core Product Terms

- Workspace -> Рабочее пространство
- Member -> Участник
- Invite -> Приглашение
- Owner -> Владелец
- Admin -> Администратор
- Integration -> Интеграция

## Delivery Analytics Terms

- Lead Time -> Время до завершения
- Cycle Time -> Цикл выполнения
- Time in Status -> Время в статусе
- Idle Time -> Время простоя
- Throughput -> Пропускная способность
- WIP (Work In Progress) -> Объем задач в работе
- Flow Efficiency -> Эффективность потока
- Rework Rate -> Доля доработок
- Loop Count -> Количество циклов возврата
- Reopen Rate -> Доля переоткрытий

## Attention Engine Terms

- Needs Attention -> Требует внимания
- Stuck Task -> Застрявшая задача
- Looping Task -> Задача с возвратами
- Inactive Task -> Неактивная задача
- Overdue Task -> Просроченная задача
- Severity -> Уровень критичности
- Suggested Action -> Рекомендованное действие

## Workflow Mapping Terms

- Not Started -> Не начато
- Ready -> Готово к работе
- In Progress -> В работе
- Review -> Ревью
- QA -> Тестирование
- Blocked -> Заблокировано
- Done -> Выполнено
- Cancelled -> Отменено
